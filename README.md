# Fraud Detection in Healthcare using Machine Learning 

 Overview
This project focuses on detecting healthcare fraud using Machine Learning. It explores the combination of "Local Outlier Factor (LOF)" and "Random Forest" algorithms for effective fraud identification 
in medical claims.

 Files
- 'data_science_project_new.pdf' — Research report and analysis
- 'healthcare_fraud_detection.ipynb' — Implementation notebook
- 'dataset.csv' — Sample healthcare claims dataset

 Techniques Used
- Local Outlier Factor (LOF)
- Random Forest Classifier
- Data preprocessing and visualization
- Fraud pattern recognition

Objectives
- Detect unusual claim patterns  
- Reduce false positives  
- Improve accuracy of fraud classification

 Tools
- Python (Scikit-learn, Pandas, Matplotlib)
- Jupyter Notebook
- GitHub
